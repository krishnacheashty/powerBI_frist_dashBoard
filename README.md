# Power BI First Dashboard Project

This repository contains my first Power BI project, showcasing data visualization and business intelligence techniques. The project focuses on transforming raw data into actionable insights through an interactive dashboard.

## 📁 Repository Contents

  * **`My_frist_dashboard.pbix`**: The primary Power BI file containing the data model, transformations, and visualizations.
  * **`My_frist_dashboard_01.pbix`**: An updated or alternative version of the dashboard.

## 🚀 Key Features

  * **Data Cleaning & Transformation**: Used Power Query to clean, format, and prepare the dataset for analysis.
  * **Interactive Visualizations**: Includes various charts (e.g., Bar charts, Line graphs, Pie charts) to represent data trends.
  * **Slicers and Filters**: Interactive elements that allow users to filter data by specific categories or timeframes.
  * **DAX Measures**: Custom calculations used to derive key performance indicators (KPIs) and metrics.

## 🛠️ Tools Used

  * **Power BI Desktop**: For creating the data model and designing the dashboard.
  * **Power Query**: For ETL (Extract, Transform, Load) processes.
  * **DAX (Data Analysis Expressions)**: For advanced data calculations.

## 📖 How to View the Project

1.  **Download Power BI Desktop**: Ensure you have [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed on your machine.
2.  **Clone the Repository**:
    ```bash
    git clone https://github.com/krishnacheashty/powerBI_frist_dashBoard.git
    ```
3.  **Open the File**: Navigate to the cloned folder and double-click on `My_frist_dashboard.pbix` to view the interactive report.
